Pixel response function of BLPP-2000 and BLPP-4000 photodetector arrays
Vaschenko P.V., Labusov V.A., Shimansky R.V.
https://doi.org/10.26896/1028-6861-2022-88-1-II-22-26


Для измерения использован лазерный нанолитограф (ЛНЛ) с X-Y системой позиционирования. 

Бескорпусные кристаллы линеек фотодетекторов **БЛПП-369М1**, **БЛПП-2000** и **БЛПП-4000** по очереди устанавливались на двухкоординатный моторизированный стол:
- Погрешность перемещения линеек составляла не более 0.1 мкм;
- Линейки перемещались вдоль линии размещения фотоячеек с шагом **0.5 мкм**, **0.5 мкм** и **0.25 мкм** для линеек БЛПП-369М1, БЛПП-2000 и БЛПП-4000 соответственно;
- На поверхность фотоячеек фокусировалось излучение непрерывного ультрафиолетового лазера с **длиной волны 405 нм**. Диаметр светового пятна в фокусе составлял **0.7 мкм**.
